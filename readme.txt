QR Code Generator ✨
A magical and modern web application to generate, download, and copy QR codes from any URL, with beautiful animations and effects.

Features
🎨 Generate QR codes from any URL
💾 Download QR codes as PNG images
📋 Copy QR codes directly to your clipboard
🧹 Clear the QR code and input easily
✨ Magical UI with animated particles, sparkles, and toasts
🖥️ Responsive design for desktop and mobile
🌐 Spanish interface
Demo
Open index.html in your browser.

Usage
Enter a URL in the input field (e.g., https://example.com).
Click "Crear Magia QR" or press Enter.
Download, copy, or clear your QR code using the action buttons.
Project Structure
index.html: Main HTML file
styles.css: All styles and animations
scripts.js: Main JavaScript logic and UI effects
Dependencies
qrcodejs (loaded via CDN)
Customization
Change colors, animations, or text in styles.css and scripts.js.
The UI and messages are in Spanish; you can translate them as needed.
